{\bfseries{Phys\+Aug\+Net}} is a research-\/grade Python toolkit designed to enhance {\bfseries{few-\/shot}} or {\bfseries{low-\/data}} industrial defect segmentation tasks by integrating\+:


\begin{DoxyItemize}
\item 🌐 {\bfseries{Vector-\/\+Quantized Variational Autoencoding (VQ-\/\+VAE)}} for {\bfseries{generative reconstructions}}
\item 🔥 {\bfseries{Physically-\/inspired augmentations}} including {\bfseries{thermal distortion}} and {\bfseries{sensor grain noise}}
\end{DoxyItemize}

This augmentation pipeline improves generalization and robustness of deep segmentation models in {\bfseries{industrial inspection}} workflows.

\DoxyHorRuler{0}
\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{🚀 Key Features}{🚀 Key Features}}\label{md__r_e_a_d_m_e_autotoc_md2}

\begin{DoxyItemize}
\item ✅ {\bfseries{Compact and efficient VQ-\/\+VAE}} architecture with fast convergence ~\newline

\item ✅ Dual-\/mode augmentation\+: {\bfseries{thermal distortion}} + {\bfseries{sensor grain noise}} ~\newline

\item ✅ {\bfseries{Phys\+Aug\+Net Fusion}}\+: Combines VQ-\/\+VAE reconstructions with physics-\/inspired augmentations ~\newline

\item ✅ CLI-\/based experiment control via {\bfseries{YAML configuration}} ~\newline

\item ✅ Fully {\bfseries{modular}} and easy to plug into Py\+Torch pipelines ~\newline

\item ✅ Lightweight design tailored for {\bfseries{few-\/shot learning}} and {\bfseries{resource-\/constrained environments}}
\end{DoxyItemize}
